## 🚀 QUICK START GUIDE

### Step 1: Open the Portal
1. Navigate to the `frontend-ui` folder
2. Open `index.html` in your web browser
3. You'll see the landing page

### Step 2: Login
Click **"Student Login"** button

**Test Credentials:**
- Email: `john@email.com`
- Password: `password123`

Or register a new account directly on the login page.

### Step 3: Explore Each Page

#### 📄 Landing Page (`index.html`)
- View college information
- See placement statistics
- Browse top recruiters
- Click "Explore Jobs" or "Get Started"

#### 👤 Student Dashboard (`dashboard.html`)
- View your profile completion status
- See recent applications
- Check notifications
- Access quick actions

#### 🏢 Browse Jobs (`jobs.html`)
- Search jobs by company/role
- Filter by package, location, type
- View job details
- Apply and save jobs

#### 👨‍💼 Your Profile (`profile.html`)
- Complete personal information
- Add academic details
- Upload resume and certificates
- Add skills
- Link social profiles

#### 📋 Track Applications (`applications.html`)
- View all your applications
- Track status (Applied → Shortlisted → Interview → Selected)
- Download application PDFs
- Schedule interviews
- View offers

#### ⚙️ Admin Dashboard (`admin.html`)
Use admin credentials:
- Email: `admin@college.edu`
- Password: `admin123`

Then access:
- Dashboard with stats
- Student management
- Job postings
- Application management
- Reports generation

## 💡 Key Features to Try

### ✨ Form Validation
Fill in any form and try:
- Enter invalid email → See error
- Short password → Password strength indicator
- Submit incomplete form → Button disabled

### 🎨 Responsive Design
1. Open any page
2. Press `F12` (DevTools)
3. Click device toggle (mobile icon)
4. Watch layout adapt automatically

### 📱 Mobile Experience
Open on your phone:
- Sidebar becomes hidden (toggle button)
- Forms stack vertically
- Buttons become full-width
- Navigation optimized for touch

### 🔔 Notifications
- Click any button with action
- See toast notification appear
- Automatically disappears after 3 seconds

### 💾 Session Persistence
- Login once
- Refresh the page
- You're still logged in (LocalStorage)
- Click "Logout" to clear session

- Select application

- Go to "Reports" tab
- Click report type
